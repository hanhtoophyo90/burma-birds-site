# Burma Birds Portal

A static biodiversity portal for the birds of Burma.

## Includes
- Homepage with hero, stats, filters, species cards, and data table
- Identify tab to create a preliminary bird identification
- Observations tab to add field records
- Documents tab to upload and store documents in the browser
- Species tab that receives records from identify and observation workflows
- Support page with funding and contact options

## How it works
This project is a front-end prototype. Data is stored in the browser using `localStorage`.

## File structure
- `index.html` — main portal page
- `support.html` — support and donation page
- `style.css` — shared visual styles
- `script.js` — tab switching, forms, storage, rendering
- `data.js` — bird dataset
- `funding.yml` — GitHub funding metadata
- `LICENSE` — MIT license

## Local use
Open `index.html` in a browser.
